## Site de Advocacia
Este projeto foi desenvolvido como atividade acadêmica com o objetivo de criar um site institucional para um escritório de advocacia, utilizando HTML e CSS.

## Objetivo:
Aplicar na prática conceitos básicos de desenvolvimento web, como:
- Estruturação de páginas com HTML  
- Estilização com CSS  
- Organização de arquivos e pastas  

## Tecnologias utilizadas
- HTML5  
- CSS3
