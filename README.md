## Site de Advocacia
Este projeto foi desenvolvido como atividade acadêmica com o objetivo de criar um site institucional para um escritório de advocacia, utilizando HTML e CSS.

## Objetivo:
Aplicar na prática conceitos básicos de desenvolvimento web, como:
- Estruturação de páginas com HTML  
- Estilização com CSS  
- Organização de arquivos e pastas  

## Tecnologias utilizadas
- HTML5  
- CSS3  

## Estrutura do projeto
site-advogado/
│
├── css/ # Arquivos de estilo
├── favicon/ # Arquivos com o icone do stie
├── imagens/ # Imagens utilizadas
│
├── index.html # Página inicial
├── equipe.html # Página da equipe
├── sobrenos.html # Página sobre o escritório
├── login.html # Página de login
├── cadastrados.html # Página de cadastro/dados
│
└── README.md

## Como executar
1. Baixe ou clone o repositório
2. Abra o arquivo `index.html` em um navegador
